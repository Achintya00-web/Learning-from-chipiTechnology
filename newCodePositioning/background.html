1. background-attachment: fixed

The background image stays completely static relative to the screen viewport while you scroll down the page.

2. background-attachment: scroll (Default)

The background image is tied to the element itself. It scrolls along with the rest of the main page.

3. background-attachment: local

Scroll inside this specific box!

The background image is anchored to the inner text content, so it scrolls up and down inside this local container.
